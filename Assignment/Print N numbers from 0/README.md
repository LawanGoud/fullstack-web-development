## 🧩 **Problem Name:**

**Print N Numbers from 0**

---

## ❓ **Goal:**

Write a program that:
✅ Reads a number `N`
✅ Prints `N` numbers starting from `0`
✅ One number per line

---

## 🧠 **Concepts Used:**

- `input()`
- `int()`
- `while` loop
- `print()`
- Counting from 0 using a loop

---

## ✅ **Code with Explanation:**

```python
# Read the number N
n = int(input())

# Start from 0
number = 0

# Repeat N times
while number < n:
    print(number)
    number = number + 1
```

---

## 🧪 **Sample Input:**

```
2
```

## 🧾 **Sample Output:**

```
0
1
```
