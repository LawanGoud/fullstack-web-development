## 🧩 **Problem Name:**

**Day Name**

---

## 🎯 **Goal:**

Write a program that:

- ✅ Reads a day number (1–7)
- ✅ Prints the corresponding day name

---

## 🧠 **Concepts Used:**

- Input/output
- `if-elif-else` conditional statements
- Mapping values using logic

---

## ✅ **Code with Comments:**

```python
# Read the day number from input
day_number = int(input())

# Check and print the corresponding day name
if day_number == 1:
    print("Monday")
elif day_number == 2:
    print("Tuesday")
elif day_number == 3:
    print("Wednesday")
elif day_number == 4:
    print("Thursday")
elif day_number == 5:
    print("Friday")
elif day_number == 6:
    print("Saturday")
elif day_number == 7:
    print("Sunday")
else:
    print("Invalid day number")  # Handles input not between 1–7
```

---

## 🧪 **Sample Input:**

```
3
```

## 🧾 **Sample Output:**

```
Wednesday
```
