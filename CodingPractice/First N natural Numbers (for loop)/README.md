🧩 **Problem Name:**  
Print N Natural Numbers (For Loop)

❓ **Goal:**  
✅ Read a number N  
✅ Print the natural numbers from 1 up to N, one number per line

🧠 **Concepts Used:**

- `input()`
- `int()`
- `for` loop
- `range()`
- `print()`

✅ **Code with Explanation:**

```python
# Read the number N
N = int(input())

# Print numbers from 1 to N (inclusive)
for i in range(1, N + 1):
    print(i)
```

🧪 **Sample Input:**

```
7
```

🧾 **Sample Output:**

```
1
2
3
4
5
6
7
```
