🧩 **Problem Name:**  
Print N Integers (For Loop)

❓ **Goal:**  
✅ Read a number N  
✅ Print the integers from 1 up to N, one integer per line

🧠 **Concepts Used:**

- `input()`
- `int()`
- `for` loop
- `range()`
- `print()`

✅ **Code with Explanation:**

```python
# Read the number N
N = int(input())

# Print numbers from 1 to N (inclusive)
for i in range(1, N + 1):
    print(i)
```

🧪 **Sample Input:**

```
3
```

🧾 **Sample Output:**

```
1
2
3
```
